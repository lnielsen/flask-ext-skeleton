Installation
============

Install Flask-YourExt with ``pip`` ::

    pip install flask-yourext

The development version can be downloaded from `its page at GitHub
<http://github.com/inveniosoftware/flask-yourext>`_. ::

    git clone https://github.com/inveniosoftware/flask-yourext.git
    cd flask-yourext
    python setup.py develop
    source run-tests.sh

Requirements
------------
Flask-YourExt has the following dependencies:

* `Flask <https://pypi.python.org/pypi/Flask>`_
* `six <https://pypi.python.org/pypi/six>`_

Flask-YourExt requires Python version 2.6, 2.7 or 3.3+
